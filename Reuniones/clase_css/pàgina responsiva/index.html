Aprendizaje Basado En Teoría Y Práctica

Sin duda, uno de los lenguajes de programación web más utilizados. Se utiliza para programar aplicaciones y páginas web dinámicas que requieran interacción con una base de datos. PHP no genera el HTML, sino que envía texto directamente al HTML de las páginas web.

Este es uno de los lenguajes de backend más sencillos de aprender debido principalmente a su sintaxis. Además, es invisible para el navegador y el cliente, ya que estos verán el HTML generado por el PHP en el servidor en función de la información que estén solicitando

Contenidos Actualizados 100%

La forma de dar vida a una página web ha cambiado bastante a lo largo del tiempo. Hace años, cualquiera podía crear una página sencilla desde un simple bloc de notas. Hoy en dia existen comppletos IDEs de programación que nos guían durante el proceso, nos ayudan a depurar el código... e incluso plataformas que nos permiten crear webs sin necesidad de usar código.

A grandes rasgos, lo único que necesitaremos para crear una página web es un editor de texto. Ya sea un bloc de notas, una alternativa como Notepad++, o un IDE como Visual Studio Code, necesitamos un lugar donde escribir. El frontend lo haremos en HTML y CSS, mientras que el backend podremos programarlo en el lenguaje que querramos.
